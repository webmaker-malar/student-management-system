# Student Management System

## About the Project

This is a simple student management system built using Python and MySQL. I created this project to understand how real applications store and manage data using a database.

## Features

* Add student details
* View all students
* Update student information
* Delete student records
* Search student by ID

## Tech Stack

* Python
* MySQL

## What I Learned

Through this project, I learned how to connect Python with a database, perform CRUD operations, and handle user input in a simple application.

## Future Improvements

* Add a user interface
* Improve the output format
* Add more validations and features

This project is a basic step towards building real-world applications using Python and databases.
